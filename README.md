## `"Часто задаваемые вопросы" (Frequently Asked Questions)`

В данной части представлена информация о разделе "Часто задаваемые вопросы" на веб-сайте. Здесь предоставляется ответы на популярные вопросы клиентов.

### Содержимое секции с вопросами и ответами:

В этом разделе ответы на несколько часто задаваемых вопросов. Каждый вопрос представлен как аккордеон:

- **Заголовок вопроса:** Это кнопка, при нажатии на которую открывается ответ на вопрос.
- **Ответ на вопрос:** Это текстовое поле, которое раскрывается, когда пользователь нажимает на соответствующий заголовок вопроса. Ответ может содержать текстовое описание, список или любую другую информацию.

Для реализации раскрывающихся ответов я использовал JavaScript для добавления/удаления классов и установки высоты элемента в зависимости от активного состояния.
```javascript
// Получение списка всех заголовков вопросов
const accordeonTitles = document.querySelectorAll(".accordeon-title");

// Добавление обработчика события для каждого заголовка
accordeonTitles.forEach.call(accordeonTitles, (accordeonTitle) => {
  accordeonTitle.addEventListener("click", function () {
    // Получение родительского элемента текста ответа
    const currentText = accordeonTitle.parentElement.querySelector(".accordeon-text");

    // Переключение классов для активного заголовка и видимости текста
    accordeonTitle.classList.toggle("accordeon-title-active");
    currentText.classList.toggle("accordeon-text-visible");

    // Установка высоты текста в зависимости от его видимости
    if (currentText.classList.contains("accordeon-text-visible")) {
      currentText.style.maxHeight = currentText.scrollHeight + "px";
    } else {
      currentText.style.maxHeight = null;
    }
  });
});

```
